# Neon Sci-Fi Calculator 🚀

A sleek, futuristic calculator built with a neon-inspired aesthetic. This project features a dynamic gradient background and a glassmorphism design.

## Features
* **Sci-Fi UI:** Uses the 'Orbitron' font and neon glow effects for a high-tech feel.
* **Animated Background:** A smooth 15-second linear-gradient transition.
* **Glassmorphism:** Semi-transparent calculator body with backdrop-filter blur.
* **Core Math:** Handles addition, subtraction, multiplication, and division.

## Tech Stack
* **HTML5**
* **CSS3** (Flexbox, CSS Animations, Grid)
* **JavaScript** (ES6)

## How to Run
1. Clone this repository: 
   `git clone https://github.com/mehrajkumarkiri/Calculator-Application.git`
2. Open `index.html` in any modern web browser.
